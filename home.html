Avertissement :

Vous trouverez dans ces quelques pages le projet de deux étudiants en développement web, élaboré pour l'ADRAR dans un cadre pédagogique.
Le site ne représente en aucun cas un produit fini, mais est un exercice visant a attester de la bonne utilisation des techniques suivantes :

  • Placement complexe (grid/flex)
  • Placement simple (margin/padding)
  • Implémentation d'images
  • Implémentation vidéo/audio
  • Balises sémantiques (nav, header, footer)
  • Sélecteurs css
  • Responsive Design (@media)

Bienvenue sur le site SHORYU NOT SHORYU !

Ce site a pour intérêt de présenter quelques jeux et éléments que les auteurs ont apprécié durant leur pratique du jeu. L'idée est de partager simplement l'un de leurs centres d'intérêt tout en mettant en pratique les techniques de développement web demandées.

Contenu :

Le site se compose en 4 parties :

  • La page d'accueil (duh)
  • Une page type "blog" présentant quelques jeux et où se les procurer
  • Une page type "gallerie" contenant quelques jolies images pour le plaisir des yeux
  • Une page type "formulaire"
